# Visual Style Guide: The Medical Audit

## 1. Typography
- **Headings:** Should always be in Title Case.
- **Emphasis:** Use **bold** for key names/entities and *italics* for direct quotes.

## 2. Component Standards
- **Alerts:** Use blockquotes for critical warnings.
> [!IMPORTANT]
> This section contains sensitive medical data.

- **Status Badges:**
- `[OPEN]` - Investigation in progress.
- `[RESOLVED]` - Systemic issue addressed.
- `[FLAGGED]` - High-priority public concern.

## 3. Image Standards
- All images must be stored in `assets/images/`.
- Use a standard naming convention: `YYYY-MM-DD-short-description.png`.
